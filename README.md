# Startup Landing Page 🚀

> A pixel-perfect, fully responsive startup marketing website with smooth animations and interactive UI components.

🔗 **[Live Demo](https://startup-3-smoky.vercel.app)**  

---

## ✨ Features

- 🎯 **Hero Section** — bold headline with clear call-to-action
- 📱 **Fully Responsive** — seamless experience on mobile, tablet, and desktop
- ✨ **Smooth Hover Effects** — interactive UI components with CSS transitions
- 🧩 **Clean Sections** — features, about, and contact sections
- ⚡ **Zero Dependencies** — pure HTML, CSS, and vanilla JavaScript

---

- Vanilla JS for interactive UI and smooth scroll behavior
- Custom hover animations using CSS transitions
- Organized codebase split into `css/`, `js/`, and `images/` folders

## 🚀 Run locally

```bash
# Clone the repo
git clone https://github.com/FrontendDevSQ/startup-landing-page.git

# Open in browser
cd startup-landing-page
open index.html
```
No installs. No build step. Just open and view.
